# react-lerna-sample

## やりたいこと

- Reactのアプリケーションとコンポーネントの開発を分離したい
- 複数のアプリケーションでReactコンポーネントを共有したい
- 共有コンポーネントへの依存を多少緩く管理したい
  - アプリAでは1.0.0、アプリBでは1.1.0、アプリCではmasterとか
  - アプリAはReact v15.2.0でアプリBはReact v.15.3.1
- 共有コンポーネントのリストをReact Storybookでホストしたい
